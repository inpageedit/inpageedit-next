## 你知道吗

- 版本代号并不具备任何现实意义，它们只是一种象征。它们也许会显示在 InPageEdit NEXT 的某些地方，但你不必过于在意它们。大家在沟通时多半也是直接称呼版本号。
- 为 InPageEdit NEXT 创建版本代号也是为了尊重 InPageEdit-v2 的习惯，v2 会给每一个版本分配一个作品《游戏人生 No Game No Life》的角色名称作为版本代号。
- 这些按首字母顺序排列的鱼类名称灵感来自于 Android 的版本代号，它们是按首字母顺序排列的糕点名字。

[^salmon]: “三文鱼”一词在中文里通常泛指可食用的鲑科鱼（尤其是大西洋鲑和部分太平洋鲑），并非严格的分类学名称。
