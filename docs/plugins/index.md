# InPageEdit NEXT 插件中心

InPageEdit NEXT 提供了强大的插件系统，让你可以轻松扩展功能。

## 内置插件

InPageEdit NEXT 自带了一系列核心插件，覆盖编辑、差异对比、页面管理等常用功能。

:point_right: [查看内置插件文档](/plugins/official/)

## 第三方插件

社区开发者们贡献了许多实用的第三方插件，你可以通过内置的[插件商店](/plugins/official/plugin-store)直接安装使用。

- [官方第三方插件列表](/plugins/thirdparty/) — 浏览社区贡献的插件
- [贡献你的插件](/plugins/thirdparty/contributing) — 将你的作品上架到官方注册表
- [自建注册表](/plugins/thirdparty/self-hosted) — 搭建私有插件源

## 插件开发

想要开发自己的插件？查看我们的[开发文档](/development/)了解详细的插件开发指南。
